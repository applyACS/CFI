instalare

  • Pune aplicația pe un server web.
  • Rulează database migrations: "php artisan migrate".
  • Fișierul .env.example este exemplul pentru datele necesare conexiunii cu baza de date. Se crează fișierul .env cu datele bazei de date.

  • Se ruleaza din folderul "public", "index.php".

  • aplicație dezvoltată cu laravel
